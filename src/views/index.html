Что такое OneScript

Этот проект является альтернативной реализацией виртуальной машины, исполняющей скрипты на языке 1С:Предприятие. При этом проект является полностью независимым от библиотек 1С и не требует наличия системы 1С:Предприятие на целевой машине.

Что такое vHttpServer

vHttpServer библиотека классов OneScript, предназначенная для организации однопоточного http сервера

Зачем это нужно

Для http сервера сожно найти массу применений, от организации удаленного доступа к заказам, до разработки внутренних сайтов. Лично я использовал его для организации удаленного доступа руководителя с планшета к отчетам в 1С 7.7.
